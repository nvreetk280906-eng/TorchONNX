# 🏠 House Price Predictor — PyTorch + ONNX + GitHub Pages

A deep neural network trained on the **California Housing dataset** that predicts median house values. Built with PyTorch, exported to ONNX, and deployed as a static website on GitHub Pages.

---

## 📁 Files

| File | Purpose |
|------|---------|
| `house_price_regression.ipynb` | Google Colab notebook — trains the model & exports ONNX |
| `house_price_model.onnx` | Trained model weights (generated by notebook) |
| `index.html` | Website front-end |
| `index.js` | ONNX Runtime inference logic in JavaScript |

---

## 🚀 How to Run

### Step 1 — Train the model in Google Colab

1. Open `house_price_regression.ipynb` in [Google Colab](https://colab.research.google.com)
2. Run all cells from top to bottom
3. The last cell will automatically download `house_price_model.onnx`

### Step 2 — Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `house-price-predictor`)
2. Upload these four files to the repo root:
   - `index.html`
   - `index.js`
   - `house_price_model.onnx`
3. Go to **Settings → Pages → Source → Deploy from branch → main**
4. Your site will be live at: `https://<your-username>.github.io/house-price-predictor/`

## 🧠 Model Details

- **Dataset:** California Housing (sklearn built-in, 20,640 samples, 8 features)
- **Task:** Regression — predict median house value in $100k units
- **Architecture:** 8 → 64 → 32 → 16 → 1 (deep neural network)
- **Activations:** ReLU on hidden layers, none on output
- **Regularization:** Dropout (0.2) on first two hidden layers
- **Loss:** MSE (Mean Squared Error)
- **Optimizer:** Adam (lr=0.001)
- **Epochs:** 150, Batch size: 32
- **ONNX opset:** 15

### Input Features

| Feature | Description |
|---------|-------------|
| MedInc | Median income in block group ($10k) |
| HouseAge | Median house age in block group |
| AveRooms | Average number of rooms per household |
| AveBedrms | Average number of bedrooms per household |
| Population | Block group population |
| AveOccup | Average number of household members |
| Latitude | Block group latitude |
| Longitude | Block group longitude |

### Output
Predicted **median house value** in $100,000s.  
*(e.g. model output of `2.5` → predicted price of ~$250,000)*

## ⚠️ ONNX Version Note

The notebook exports with `opset_version=15`.  
The website loads `onnxruntime-web 1.16.3` from CDN, which is compatible with opset 15.  
If you change the opset version in the notebook, update the CDN script tag in `index.html` to match.
